# Entorno n8n (sin instalación) — Codespace Github

## Instrucciones
1. Entra al repositorio
2. Code → Codespaces → Create codespace on main
3. Cuando cargue VS Code web, abre la pestaña **Ports**
4. Abre el puerto **5678** → se abrirá n8n

## Workflow de ejemplo
Crear un sistema Helpdesk del aula:
- Alta de incidencia (Webhook)
- Registro en Google Sheets
- Email al profesor + confirmación al alumno
- Regla URGENTE
- Cierre de incidencia

## Entrega (solo para alumnos SMX1 - FPLLEFIA)
- Exporta los workflows: **Workflow → Download**
- Entregar:
  - workflow1.json
  - workflow2.json
  - 4–6 capturas (webhook, sheet, emails, cierre)
  - mini memoria (½–1 página)

Fecha | Nombre | Grupo | Tipo | Mensaje | Email | Estado
